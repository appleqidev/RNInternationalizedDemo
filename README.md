# RNInternationalizedDemo
一个react native 多语言(国际化)demo
## 安装

 - `git clone https://github.com/giantss/RNInternationalizedDemo.git`
 
## 初始化
  `npm install`
  
## 运行
  `react-native run-android/run-ios`
